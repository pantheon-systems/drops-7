Simple module to add a checkbox in the menu section of the node add/edit form to toggle visibility of a nodes menu item.

Installation
------------
Copy the disable_node_menu_item module into your module directory and then enable on the admin modules page.

Usage
-----
Once the module has been installed navigate to admin/structure/menu/dnmi
Here you are able to select which content types you would like to include the functionality provided by this module on
Once you have selected the content types you would like to add this functionality to a checkbox will be added to node add/edit forms
of the content types that you have selected in the configuration.
